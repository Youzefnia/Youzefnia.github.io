# Arash Youzefnia — Portfolio

## What this is
A personal portfolio site for Arash Youzefnia, a Medical Writer & Research Associate (medical biotechnology background) who is pivoting into building apps and websites with AI ("vibe coding"). The site exists to let other people see that journey — his AI-building skills, the courses he's working through, and the projects taking shape — even though he doesn't have finished apps yet.

## Audience
People evaluating Arash as a builder/collaborator: potential clients, collaborators, course communities (Scrimba, Vanderbilt), and anyone he shares the link with directly. Visitors are likely AI-curious or already in the "vibe coding" / no-code-to-code space.

## Register
**Brand** — this is a portfolio/personal site where design and presentation are part of the message (per impeccable's register split: marketing/portfolio = brand, design IS the product). `reference/brand.md` was not available on this install; treat this note as the substitute until that file exists locally.

## Goals
1. Establish credibility despite having no finished coding projects yet — lean on real, verifiable background (medical writing, research, AI/prompt-engineering work) rather than overstating coding experience.
2. Make the "currently learning" story (two specializations) feel like active momentum, not an excuse.
3. Keep unfinished projects honest — labeled "Coming Soon" / "In Progress", linked to live GitHub repos, not faked with placeholder screenshots.
4. Easy path to contact (email, GitHub) for anyone interested in following along or collaborating.

## Structure
- **Hero** — name + one-line positioning ("Medical Writer & Research Associate — now learning to build apps and websites with AI")
- **About** — bio bridging scientific/writing background to AI-assisted building, plus a quick-facts panel (location, background, core skills, now-learning)
- **Journey** (Currently Learning) — two in-progress specializations (Scrimba "Vibe Coding Essentials", Vanderbilt "Generative AI Software Engineering"), plus a list of already-completed adjacent certs (Google Prompting Essentials, HubSpot SEO I & II)
- **Projects** — MetaHealth and MD-Simiulation-Analysis, both early-stage GitHub repos, shown as "Coming Soon" cards linking out to GitHub, plus a ghost card pointing to the GitHub profile for more
- **Contact** — email (youzefnia@gmail.com) and GitHub (github.com/Youzefnia). No LinkedIn, by request.
